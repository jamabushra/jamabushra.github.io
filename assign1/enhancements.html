Our Enhancements

FULLY RESPONSIVE WEBSITE

The enhancement lets the website be fully responsive and easily viewable on both a mobile device
aswell as bigger screens with relevant adjustments made for the different screen sizes

The code for the enhancement is as follows:

@media only screen and (min-width: 1000px) {
header .header-brand {
margin: 31px 0;
text-align: left;
line-height: 72px;
padding: 0 20px 0 237px;
border-right: 7px solid #001F67;
float: left;}
header nav ul {
margin: 30px 0px;
float: left;
}
header nav ul li a {
line-height: 60px;
}
}

Source of inspiration:

https://www.youtube.com/watch?v=kbLfWKGVsMQ followed and coded along tutorial to implement advanced
features such as mobile responsiveness to website

To see enhancement in action - Just resize screen

EXTRA HTML & CSS NOT COVERED IN READING MATERIAL

animations done on hover using flexbox attributes also added a wrapping attribute to make images more
responsive and flexible to various screen sizes

The code for it is as follows:

.container {
display: flex;
flex-wrap: wrap;
justify-content: center;
}
.container img {
width: 450px;
height: 300px;
margin: 7px;
transition: all 1s; /*this*/
}
.container img:hover {
transform: scale(1.1);/*coupled with this makes it scale for 1 second when i hover cursor over image*/
}

Source of inspiration:

https://www.udemy.com/the-complete-web-developer-zero-to-mastery/ followed along and coded with section on how to animate css using flexbox
